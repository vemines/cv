Tổng Quan

Một lập trình viên full-stack đam mê xây dựng ứng dụng đa nền tảng. Thành thạo Flutter trong việc phát triển giao diện di động responsive và clean architecture kết hợp với backend Firebase. Có kinh nghiệm phát triển RESTful APIs cho backend sử dụng Node.js và các cơ sở dữ liệu phổ biến như MongoDB, MySQL. Ngoài ra, tôi cũng phát triển ứng dụng web với ReactJS.

Kỹ Năng

Flutter: Có kinh nghiệm trong việc xây dựng ứng dụng di động với backend Firebase. Thành thạo sử dụng các công cụ quản lý trạng thái (Bloc, GetX, Provider), thiết kế giao diện responsive (mobile, tablet) với Material Design. Có khả năng áp dụng và tùy chỉnh các thư viện của bên thứ ba.

Node.js: Có kinh nghiệm xây dựng RESTful API với Express.js, routing, middleware và HTTP method. Cơ sở dữ liệu MongoDB, MySQL, Redis caching và các dịch vụ bên thứ ba như AWS (EC2, S3, Cloudfront) và Cloudinary. Có kiến thức cơ bản về microservices.

React: Thành thạo trong việc tạo giao diện web responsive với JavaScript hiện đại (ES6+). Có kinh nghiệm với functional components (useState, useEffect), routing, validate form và trang trí cho các components.

Golang: Đã hoàn thành Go Tour. Có kinh nghiệm xây dựng API RESTful với Gin và Gorm, bao gồm routing, middleware và các phương thức HTTP.

Ngôn ngữ khác: Git (quản lý mã nguồn), Python
