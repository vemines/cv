Tổng Quan

Lập trình viên Flutter đầy nhiệt huyết, có kinh nghiệm xây dựng ứng dụng di động đa nền tảng chất lượng cao. Thành thạo trong việc phát triển giao diện người dùng (UI) responsive, áp dụng kiến trúc Clean Architecture, và tích hợp với các API RESTful cũng như backend Firebase. Có khả năng xây dựng các API RESTful sử dụng Node.js, MongoDB và MySQL. Tìm kiếm cơ hội để đóng góp vào việc phát triển các giải pháp di động tiên tiến.

Kỹ Năng

Flutter: Kinh nghiệm thực tế trong việc xây dựng ứng dụng di động, bao gồm tích hợp backend Firebase. Thành thạo các giải pháp quản lý trạng thái (Bloc, GetX, Provider), thiết kế giao diện responsive (mobile, tablet) theo Material Design. Sử dụng thành thạo các thư viện phổ biến như `dio`, `dartz`, và `get_it`.

Node.js: Kinh nghiệm xây dựng API RESTful với Express.js, bao gồm routing, middleware, xác thực (authentication) và các phương thức HTTP. Thành thạo cơ sở dữ liệu MongoDB, MySQL, Redis (cho caching), và tích hợp với các dịch vụ AWS (EC2, S3, CloudFront) và Cloudinary.

Golang: Đã hoàn thành Go Tour và có kinh nghiệm xây dựng API RESTful cơ bản với Gin và Gorm. (Đang tiếp tục học hỏi và phát triển kỹ năng Golang).

Kỹ năng khác: Git (quản lý mã nguồn), Python, Docker (cơ bản)

Tiếng anh: Trình độ đọc viết trung cấp, giao tiếp cơ bản.
