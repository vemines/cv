Tổng Quan

Một lập trình viên full-stack đam mê xây dựng ứng dụng đa nền tảng. Thành thạo Flutter trong việc phát triển giao diện di động đáp ứng (responsive) kết hợp với backend Firebase. Có kinh nghiệm phát triển RESTful APIs cho backend sử dụng Node.js và các cơ sở dữ liệu phổ biến như MongoDB, MySQL. Ngoài ra, tôi cũng phát triển ứng dụng web với ReactJS.

Kỹ Năng

Flutter (Fresher): Có kinh nghiệm trong việc xây dựng ứng dụng di động với backend Firebase. Thành thạo sử dụng các công cụ quản lý trạng thái (Bloc, GetX, Provider), thiết kế giao diện responsive (mobile, tablet) với Material Design. Có khả năng áp dụng và tùy chỉnh các thư viện của bên thứ ba.

Node.js (Thực tập sinh): Có kinh nghiệm xây dựng API RESTful với Express.js, routing, middleware và HTTP method. Cơ sở dữ liệu MongoDB, MySQL, Redis caching và các dịch vụ ebn6 thứ ba như AWS (EC2, S3, Cloudfront) và Cloudinary. Có kiến thức cơ bản về microservices.

React (Thực tập sinh): Có kinh nghiệm xây dựng API RESTful với Express.js, bao gồm routing, middleware và các phương thức HTTP. Sử dụng cơ sở dữ liệu MongoDB, MySQL, Redis caching và các dịch vụ bên thứ ba như AWS (EC2, S3, CloudFront) và Cloudinary. Có kiến thức cơ bản về microservices.

Golang (Thực tập sinh): Đã hoàn thành Go Tour. Có kinh nghiệm xây dựng API RESTful với Gin và Gorm, bao gồm routing, middleware và các phương thức HTTP.

Ngôn ngữ khác: Git (quản lý mã nguồn), Python

Clone Google Keep

Clone giao diện người dùng và các tính năng của Google Keep bằng Flutter. Sử dụng Bloc và Provider làm state manager. Tích hợp backend với Firebase. Áp dụng các thư viện generate code.

Ngôn ngữ: Flutter

Tính năng: Auth, CRUD Note (với Camera, Tải ảnh lên, Bảng vẽ tay, Ghi âm)

Generator: intl_utils (ứng dụng đa ngôn ngữ), hive_generator (hỗ trợ lưu đối tượng cho bộ nhớ đệm), flutter_intl (quản lý assets), flutter_gen (tạo màu từ xml)

State manager: Bloc, Provider

Backend: Firebase, Hive

Vai trò: FullStack

Repo URL: https://github.com/vemines/clone-keep-flutter
