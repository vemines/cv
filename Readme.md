# CV Page Written with HTML and CSS

This project is a personal CV page created using HTML and CSS. It showcases my skills, projects, and work experience as a Fullstack Developer.

## Live Demo

You can view a live demo of the project at:
<br/>

[English](https://vemines.github.io/cv)
<br/>
[Vietnamese](https://vemines.github.io/cv/index-vi.html)

## Features

- Responsive design that adapts to different screen sizes.
- Interactive elements that allow users to copy contact information to the clipboard.
- A section for showcasing skills, projects, and work experience.

## Technologies Used

- HTML5
- CSS3
- JavaScript
